Context & Stakes

DIRECTV was losing 400,000 subscribers per quarter. Leadership mandated a 15% ARPU increase through premium add-ons, but three disconnected systems meant 180-day launch cycles. Miss the NFL kickoff = lose $15M. Miss holiday bundles = lose $20M.

The Challenge

Technical Blockers

  • No middleware layer
  • Campaign rules hardcoded in firmware
  • Data siloed across 3 systems
  • No A/B testing capability

Operating Model Blockers

  • 3 different intake processes
  • Tribal knowledge, zero documentation
  • No RACI or ownership clarity
  • Every launch was bespoke

Success Criteria: 180 ‚Üí 120 day launches. 3x campaign velocity. $50M+ incremental revenue.

What I Owned

Strategy & Architecture

  • Defined product vision & technical architecture
  • Designed middleware layer & API contracts
  • Prioritized roadmap across 8 teams
  • Secured $1.2M budget & exec sponsorship

Delivery & Operations

  • Led cross-functional delivery (Eng, Product, Marketing, RevOps)
  • Built operating model: intake, RACI, playbooks
  • Managed risk & seasonal dependencies
  • Shipped STB ‚Üí .com ‚Üí Mobile incrementally

Key Decision: Build middleware first, ship STB as MVP, validate the model, then roll out .com and mobile. This de-risked delivery and proved value early.

The Solution

Technical Solution

  • Middleware platform unified offer logic across channel implementations
  • Centralized offer database eliminated manual syncing
  • REST APIs gave STB, .com, and mobile a consistent interface
  • Configuration-driven rules engine enabled campaigns without code changes

Operating Model

  • Single intake form created one source of truth for campaign execution
  • Weekly grooming aligned delivery planning to the sports calendar
  • RACI framework, playbooks, and templates reduced handoff friction
  • Post-launch reviews improved throughput and repeatability across teams
